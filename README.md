# 코리아IT아카데미 수강료 견적 시스템

교육 상담사가 수강생에게 실시간 수강료 계산 및 견적서를 발급하는 웹 앱입니다.

## 기능
- 9가지 과정 유형별 과목 선택 (멀티 체크박스)
- 수강료 직접 조정 가능
- 통합 할인 슬라이더 (0~100%, 5% 단위)
- 할인 명분 3개 입력 (엑셀 출력 시 자동 매핑)
- 상담자 정보 입력 (이름/연락처)
- 엑셀 견적서 즉시 다운로드 (SheetJS)

## 로컬 실행
`index.html` 파일을 브라우저에서 바로 열면 됩니다.

## GitHub Pages 배포
1. 이 폴더를 새 GitHub 저장소로 push
2. Settings → Pages → Branch: main / folder: / (root)
3. 저장 후 배포 완료 (약 1분 소요)

## 파일 구조
```
index.html  ← 전체 앱 (HTML + CSS + JS 단일 파일)
README.md
```
